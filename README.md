# Build Your Own LoRaWAN Sensor With SenseCAP S2110 Grove to MODBUS RS485 Converter

This is the source code for [SenseCAP S2110 Grove to MODBUS RS485 Converter](https://www.seeedstudio.com/SenseCAP-XIAO-LoRaWAN-Controller-p-5474.html) and it provides an open-source tool to build your own LoRaWAN sensors.

## Hardware Set Up

- **Step 1:** Unscrew the four screws on the SenseCAP S2110 Converter lid to open the lid and unscrew the PG connector (with a coin)

<div align=center><img width=1000 src="https://files.seeedstudio.com/wiki/SenseCAP-S2110/92.jpg"></div>

- **Step 2:** If you have already made the connection between the SenseCAP S2110 Converter and the Data Logger with the 4 wires, it is recommended to remove the red wire which connects to the 5V pin of the SenseCAP S2110 Converter

<div align=center><img width=1000 src="https://files.seeedstudio.com/wiki/SenseCAP-S2110/97.jpg"></div>

- **Step 3:** Connect one end of a USB Type-C cable to the SenseCAP S2110 Converter and the other to the PC

<div align=center><img width=1000 src="https://files.seeedstudio.com/wiki/SenseCAP-S2110/102.png"/></div>

Now you can open [Arduino IDE](https://www.arduino.cc/en/software) on the PC and start tinkering!
